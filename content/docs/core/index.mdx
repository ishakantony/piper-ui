Piper provides a platform to run workload known as **pipelines**. One you got a pipeline, you **start** it, do your thing, **stop** it.

## Features

It comes with the following features out of the box:

1. Create and manage pipelines.
2. Create and manage pipeline templates.
3. Create and manage pipeline instances.
4. Dashboard (_in progress_).
5. Statistics (_in progress_).
